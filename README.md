# roulette-wheel

## Description

Application used for entertainment. It helps to decide if we are not determined what should we choose, do or eat etc. 

## Features

* `adding new elements`
* `chaning color of elements`
* `remove single element`
* `remove all elements`
* `dark and light mode`
